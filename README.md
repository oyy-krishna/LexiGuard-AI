## ✨ Features

- **📄 Universal Document Ingestion**: Upload local `.pdf` / `.docx` files or directly paste remote `PDF URLs`.
- **🔍 Smart Retrieval (RAG)**: Built with LangGraph orchestration and ChromaDB local vector storage to maintain exact metadata and accurate search scope.
- **🛡️ Legal-Grade Accuracy**: Integrates a `Search` ➡️ `Analyze` ➡️ `Verify` ➡️ `Respond` LangGraph node flow to ensure strict legal objectivity and precise page-level source citations.
- **🎨 Glassmorphism Meets Dark Mode**: Streamlit-based UI customized with an ultra-premium dark theme, glowing UI borders, and frosted glass components.
- **⚡ Real-Time Streaming**: Watch exactly what the AI is thinking as it updates its internal state statuses (e.g., *“Analyzing Clauses...”*), followed by real-time typing of the final insight.

## 🛠️ Tech Stack

- **Frontend Strategy:** [Streamlit](https://streamlit.io/) + Custom CSS
- **Orchestration:** [LangGraph](https://python.langchain.com/v0.1/docs/langgraph/) & LangChain
- **LLM Engine:** HuggingFace `HuggingFaceH4/zephyr-7b-beta`
- **Vector Database:** [ChromaDB](https://www.trychroma.com/)
- **Data Ingestion:** [PyMuPDF4LLM](https://pymupdf.readthedocs.io/en/latest/) for high-fidelity text/table OCR extraction

## 🚀 Getting Started

Follow these steps to get LexiGuard AI running on your local machine.

### 1. Prerequisites
- **Python 3.9+** installed on your machine.
- A **HuggingFace API Token** (`HF_TOKEN`).

### 2. Installation Setup

Clone the repository and install the required dependencies:

```bash
# Clone the repository
git clone https://github.com/oyy-krishna/LexiGuard-AI.git

# Navigate into the project directory
cd LexiGuard-AI

# (Optional but recommended) Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate

# Install the dependencies
pip install -r requirements.txt
```

### 3. Environment Variables

You must export your HuggingFace API key for `langchain-huggingface` to locate it successfully:

**For MacOS/Linux:**
```bash
export HF_TOKEN="hf_your-huggingface-token-here"
```

**For Windows (PowerShell):**
```powershell
$env:HF_TOKEN="hf_your-huggingface-token-here"
```

### 4. Run the Application

Launch the Streamlit app:

```bash
streamlit run app.py
```

The application will open automatically in your default internet browser targeting `http://localhost:8501`.

## 📖 How to Use

1. **Accept the Disclaimer**: Upon launching, explicitly review and accept the protective Legal Disclaimer popup.
2. **Choose Source Input**: Use the glassmorphism Sidebar to select either **Upload File** or **PDF URL**.
3. **Process Source**: Hit the "Process Button", wait a few moments as PyMuPDF digests your document formats and encodes them directly into your local Chroma VectorStore.
4. **Chat**: Using the bottom chat input, ask detailed legal queries. (e.g., *"What is the term length and termination clause for this rental agreement?"*). Wait for the multi-step status engine to complete its graph nodes and real-time stream the result to your screen.
